# Stealthburner Nekko RMX
Stealth Burner nekko ear type for BMG with filament sensor and LGX Light for CW2

- 1x PC4 M6 fitting
- 1x 7mm steel ball
- 1x Omron D2F-L
- 2x Jst 2.54 connector for soldered that lead of Omron D2F_L microswitch
